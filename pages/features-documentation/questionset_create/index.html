Selecting Questions - Using filter

Step Screen
1. Type keywords on the Search Bar to search the repository for questions. The search results are displayed in the bottom left pane
2. Select Advanced Filters for better search result
 a) Language
 b) Difficulty
 c) Question Type
 d) Grade
 e) Concepts
3. Use the My Questions slider to search for questions that you have uploaded. To search for questions available on the portal, disable this option
4. The search result is displayed on the left pane
5. The selected question can be previewed on the right pane
6. Select questions from the results displayed by marking the checkbox
7. Click Create Question to create your own question. Click Create Question to know more.
8. Click Next

Creating Questions

Step Screen
To create a question set
1.Select the type of question you want to create
2. Click Select
3. Enter the questions in Question Configuration page.
 a) If you have selected Multiple choice question, refer Creating Multiple Choice Question Set
 b) If you have selected Fill in the blanks questions, refer Creating Fill in the blanks Question Set
 c) If you have selected Match the following questions, refer Creating Match the Following Question Set

1. Enter the Question Details
 a) The Title displays the title
 b) Select Language
 c) Select Level
 d) Select Grade Level
 e) Select Max Score
 f) Select Concept
 g) Enter Description

Creating Multiple Choice Question Set

Step Screen
Multiple Choice Questions
1. Enter Question Set Title for the questionaire
2. The Max Score displays the maximum score for the question
3. Select the number of questions that should be displayed in each screen from the Display drop-down menu. The question amongst the selected questions is displayed in the right pane
4. Select Show Immediate Feedback to determine when the feedback for the answers should be displayed
5. Select Shuffle Question to enable shuffling of questions from the set
6. The list of selected questions appears in the left pane
7. Enter the score for each question using Configure Score
8. Click Add More Questions to increase the questions in the question set
9. Click Add to append the selected questions in Question Set. The Question Set is added on the slide.

Creating Match the Following Question Set

Step Screen
1. Enter the question
2. Enter the answers
3. Select the answer amongst the multiple answers
4. Click Delete icon to remove an answer
5. Click Image icon to add an image
6. Click Audio icon to add an audio message
7. Select Add answer to add another option in the Mark as Right multiple choice
8. Click Refresh icon to preview your question
9. Select the layout from the drop-down menu
10. Use Shuffle Option to change the sequence of answers
11. Select Back to go back to the previous page
12. Click Next to enter the Question Details.

Customizing Question Set

Step Screen
After adding the question set, the customize tab displays options to modify them. You can:
1. Click Assessment Browser to return to the Selecting Questionsscreen
2. Click Question Config to manage the configuration of the selected questions
3. View and modify the Assessment Title, if required
4. Use the sliders to enable or disable Shuffle Questions, Shuffle Options and Show Immediate Feedback
5. Select the number of questions to be displayed in each screen
6. Enter the Total Marks for the assessment

Configuring Question Set

Step Screen
1. Click the trash icon to delete the added question set
2. Click the spanner icon to edit details of the question set. The settings options are displayed on the right
a) Click on the upward/downward arrows to determine the marks of the question set
3. Click the eye icon to view the preview of the question set
4. Click Done to save
