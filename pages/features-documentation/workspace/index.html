My Workspace

Sunbird provides you with a workspace that is only yours. Once you have your own unique profile, you can start using your workspace to create, curate, review, upload and publish different types of content artifacts using the customizable content editor. You can save your drafts, send them for a review, flag an issue, rework on them as often as you like until you publish it for use by the community.

Accessing your Workspace

Step Screen
1. Login with the registered credential, to land into Home page
2. Click Profile tab on the header
Or
3. Click Profile icon to the right

Viewing your Workspace

Step Screen
Click View your Workspace to land to My Workspace page
In your Workspace you can:
- Create Book, Course, Study material, Collection, Lesson plan
- View Drafts
- Review content submitted
- View published content
- View all uploaded content
- Limited publishing
Note:

Up For Review feature

Note: Up For Review, feature is only available for the users with Reviewer Rights

Step Screen
1. Login with registered credential and password
2. Go to My Workspace
3. Click Up For Review in your workspace to see all the contents available for review
Up For Review page opens. View details viz,
1. Content: You can see all the contents send for review
2. Author: The author name is displayed for the content created by them
3. Status: You can see the status for all contents. Reviewer can review and publish the content
