### Selecting Questions

<table>
  <tr>
    <th style="width:35%;">Step</th>
    <th style="width:65%;">Screen</th>
  </tr>  
  <tr>On the <b>Question Set</b> page, you can apply filters, search for questions and select appropriate questions here  
   <td>1. Type keywords to search for questions
     <br>2. Apply filters for better search results
      <br>&emsp;a) <b>Language</b>
      <br>&emsp;b) <b>Difficulty</b>
      <br>&emsp;c) <b>Question Type</b>
      <br>&emsp;d) <b>Grade Level</b>
      <br>&emsp;e) <b>Concepts</b>
<br>3. Use <b>My Questions</b> slider to search for questions that you have uploaded. To search for questions available on the portal, disable this option
   <br>4. Select questions from the results displayed on the left pane 
   <br>5. The selected question can be previewed on the right pane 
   <br>6. Click <b>Create Question</b> to create your own question
   <br>7. Click <b>Next</b>
</td>
  <td><img src="pages/features-documentation/images/questionset/selectquestions.png"></td>
</tr>

#### Creating Multiple Choice Question Set
 <table>
 <tr>
    <th style="width:35%;">Step</th>
    <th style="width:65%;">Screen</th>
 </tr>
 <tr>
<td>1. Enter the question
<br>2. You can format the questions using rich text editor 
<br>3. Enter the answers 
<br>4. Mark the correct answer amongst the multiple answers 
<br>5. Click <b>Delete</b> icon to remove an answer
<br>6. Click <b>Image</b> icon to add an image. Refer <a href="features-documentation/metadata_addingimages" target="_blank">Adding Images</a>
<br>7. Click <b>Audio</b> icon to add an audio 
<br>8. Select <b>Refresh</b> icon to preview your question 
<br>9. Select the layout from the drop-down menu
<br>10. Use <b>Shuffle</b> Option to change the sequence of answers
<br>11. Click <b>Next</b> to enter the Question Details. The <b>Create Question</b> page appears 
<br>12. Select <b>Back</b> to go back to the previous page
  </td>
    <td><img src="pages/features-documentation/images/questionset/multiplechoicequestion.png"></td>
 </tr>
 </table>
 
#### Creating Fill in the Blanks Question Set
  
<table>
 <tr>
    <th style="width:35%;">Step</th>
    <th style="width:65%;">Screen</th>
 </tr>  
 <tr>
 <td>1. Enter the question
 <br>2. You can format the questions using rich text editor 
 <br>3. Enter the answers within a pair of square brackets [[Answer]]
 <br>4. Use <b>Select Keyboard</b> option select the text input method for the question 
 <br>5. Select <b>Refresh</b> icon to preview your question
 <br>6. Select <b>Partial Scoring</b> to enable scoring for each correct answer
 <br>7. Click <b>Next</b> to enter the Question Details. The <b>Create Question</b> page appears
 <br>8. Select <b>Back</b> to go back to the previous page
 </td>
    <td><img src="pages/features-documentation/images/questionset/Fillintheblanks.png"></td>
 </tr>
 </table>

#### Creating Match the Following Question Set
  
<table>
 <tr>
    <th style="width:35%;">Step</th>

## Configuring Question Set
 
<table>
<tr>
   <th style="width:35%;">Step</th>
   <th style="width:65%;">Screen</th>
</tr>  
<tr>
  <td>1. Enter <b>Name</b> of the question set 
  <br>2. Enter <b>Description</b> 
  <br>3. Select the following from the drop-down menu
    <br>a. Select <b>Medium</b>
    <br>b. Select <b>Level</b>
    <br>c. Select <b>Class</b>
    <br>4. Enter the <b>Max Score</b> of the question
    <br>5. Select the <b>Concepts</b> of the question 
    <br>6. Select <b>Refresh</b> icon to preview your question
    <br>7. Select <b>Partial Scoring</b> to enable scoring for each correct answer
    <br>8. Click <b>Submit</b> to configure the question details
    <br>9. Select <b>Back</b> to go back to the previous page 
  </td>
  <td><img src="pages/features-documentation/images/questionset/questionsetcustomised.png"></td>
 </tr>
</table>
