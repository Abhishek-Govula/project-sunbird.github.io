Compiling Courses on Sunbird

On Sunbird, you can compile a course by adding content to it. You can create each unit of course content and then compile them or compile a course using existing content in the portal.

Prerequisites

Step Screen
1. You are logged in
2. You are currently on My Workspace page
3. You want to create courses
4. You have clicked the Courses tile
1. Enter Name of the course
2. Enter Description of the course
3. Click Start Creating ***Note: These details can be entered later when filling the metadata of the course***

Course Metadata

Step Screen
1. Click Update in the right navigation pane to edit details of the course
2. Enter Title of the course
3. Enter Description
4. Select the following from their respective drop-down lists

 a) Audience
 b) Language
 c) Concepts
5. Enter Faculty details
6. Enter Tutor details
7. Enter Keywords
8. Click Update to update the details

Adding Image to Course Metadata

Step Screen
Click the Image icon to add an image
1. You can add images either by selecting an image from:
 a) A list or uploading images
 b) Your local machine to use immediately
2. Click My images to view a list of all image files that you have previously,uploaded
3. Click All image to view a list of image files from the repository
Note: The repository contains all image files added and uploaded by any users
5. Choose an appropriate image file from any list and click Select
6. Click Upload and use to upload and use images from your gallery
1. Choose File to browse and click Select a file from your local drive or open the explorer in a separate window or drag and drop the required file on the screen
Note: Supported file formats are: .jpeg, .jpg, .png
2. Under Copyright and License, select if you want to:
 a) Make it available to everyone
 b) Not share with anyone
4. Add image details in the Asset Caption area
5. Click Upload and Use to upload the image

Adding Content to Course

</tr>
Step Screen
1.Click +icon in the left navigation pane to add a Course Unit, Collection, Resource, Story or Worksheet,
2.Click on the content type you want to add to the course

Adding Course Units

Step Screen
1. On clicking Course Unit from the drop-down, the metadata for course unit is displayed on the right
2. Click Update to edit the metadata
3. Click Update to save the edited metadata

Adding Collections, Resources, Stories and Worksheets

</tr>
Step Screen
1. On clicking Collections, Resources, Stories or Worksheets, the Browse Resource page is displayed
2. Type keywords to search for content
3. For faster and effective search of the lessons, you can apply filters. To apply filters, select from the drop-down lists available for Language, Grade, Content Type, and, Concept
4. Click Apply. Results are loaded on the right pane
5. Select the content you want to add and click Add to add the selected content
1. Click Save on the header to save the course
2. Click Send for review to send the created course unit for review
