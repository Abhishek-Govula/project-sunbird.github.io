Step Screen
On Sunbird you can search the content using
1.Enter the text in Search text box.
2.Click search icon
Note: You can further filter the search result
3.Enter the Enter DIAL Code text box. Refer Searching Content Using DIAL Code
4.Select following filter from their respective drop-down lists to filter the content:
 a) Board
 b) Medium
 c) Subject
 d) Class
5.Click Submit to search using filters
6.Click Reset to reset filters
7.Browse through the list of available content. Use
a.Previous to go to previous set of content
b.Next to go to next set of content
