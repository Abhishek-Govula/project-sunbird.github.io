Lesson plan is a structured outline of a given topic/unit/chapter to be taught in any given day. Lesson plan aids to streamline teaching and create an engaging learning experience.

Prerequisites

Step Screen
1. You are logged in
2. You are currently on My Workspace page
3. You want to create a lesson plan
4. You have clicked the Lesson Plan tile

Creating Lesson Plan

Step Screen
1. Enter a Name for the lesson plan
2. Select other metadata from their respective dropdowns:
a) Curriculum
b) Medium
c) Subject
d) Class
3. Click Start Creating to create the lesson plan

Adding Metadata

Step Screen
1. Enter Title
2. Enter Description for the lesson plan
3. Select the following from their respective drop-down lists:
 a) Curriculum
 b) Subject
 c) Grade
 d) Medium
 e) Language
 f) Concepts
4. Enter Notes to Teacher, if any
5. Enter Duration of the lesson plan
6. Describe the Learning Objective for the lesson plan
7. Click Lesson Plan on the left navigation pane to add other units of content to the plan
Note: You can edit or update the title of units in the collection from the navigation hierarchy
8. If you wish to use the keyboard shortcuts for collection editor, refer Keyboard Shortcuts
Note: You can add or update metadata for multiple pages within the collection and click Save to collectively save all updates.

    Adding Content to Lesson Plan

    Step Screen
    1.Click the burger menu in the left navigation pane to add content Units or Resources
    2. Alternatively, click New Child node to add a lesson unit. Any number of units can be added
    3. Click New Sibling node to add a another Unit within the created lesson unit
    Note: A new sibling node can be added only after adding a child node
    1. Click Edit to edit the lesson name
    2. Click Delete to delete the lesson unit. This option is highlighted only after adding lesson unit/resource
    3. Click New Sibling to add lesson unit. This option is highlighted only after a child note is added. A sibling unit is created as a sub-node within a child unit
    4. Click New Child to add a lesson unit
    5. Click Add Resource to add a resource to the lesson unit
    Note: You can alternatively use the keyboard shortcuts displayed or refer Keyboard Shortcuts

    Adding Lesson Plan Units

    Step Screen
    On clicking New Child from the burger menu, the metadata for lesson unit is displayed on the right
    1. Enter Unit Title
    2. Enter Description about the lesson unit
    3. Enter Keywords
    4. Click Save to save the metadata of the lesson unit
    5. Click Image icon to add image to the lessonplan unit metadata. For details on adding image, refer Adding Image to Lesson Plan Metadata

    Adding Resources

    Step Screen
    1. Click Resources from the burger menu drop-down to add resources
    Note: Resources can be added within a lesson plan or within a lesson unit. Select Resources from the respective burger menu drop-downs
    1. The Browse Resource page is displayed
    2. Type keywords to search for content
    3. For faster and effective search of the resources, you can apply filters. To apply filters, select from the drop-down lists available for:
     a) Category
     b) Language
     c) Class
     d) Subject
    4. Click Reset to edit and reset the filters
    5. Click Apply. Results are loaded on the right pane
    6. Select the content you want to add
    7. click Add to add the selected content
    8. Click Close icon to go back to the main menu

    Deleting Content

    Step Screen
    1. Click Delete icon beside the burger menu
    2. Click Yes to delete the added content
    Note: You can alternatively delete content by clicking Delete in the burger menu drop-down

    Reordering Content

    Step Screen
    1. Drag and drop the content to the desired position to rearrange the order of the content added to the lesson plan
    1. Click the upward arrow to collapse the table of contents. Click the same arrow to expand collapsed table of contents.

    Sending Lesson Plan for Review

    Before the Lesson Plan is publihed, it must be reviewed. You can send it for reviewing and publishing.

    Step Screen
    1. Click Save on the header to save the lesson
    2. Click Send for review to send the created lesson for review
    3. Click Limited sharing to publish the lesson to the limited audiences
