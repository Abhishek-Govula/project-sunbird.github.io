Lesson plan is a structured outline of a given topic, unit, chapter to be taught on any given day. Lesson plan aids to streamline teaching and create an engaging learning experience.

Prerequisites

Step Screen
1. You are logged in
2. You are currently on My Workspace page
3. You want to create a lesson plan
4. You have clicked the Lesson Plan tile

Creating Lesson Plan

Step Screen
1. Enter a Name for the lesson plan
2. Select other metadata from their respective dropdowns:
a) Board
b) Grade
c) Subject
d) Medium
3. Click Start Creating to create the lesson plan. A new Lesson Plan is created and displayed in a new page.

Adding Metadata

Step Screen
1. Click the Edit Details link on the top left corner of the page.
2. Enter Title
2. Enter Description for the lesson plan
3. Enter Keywords
4. Select the following from their respective drop-down lists:
 a) Curriculum
 b) Subject
 c) Attributions
 d) Medium
 e) Audience
 f) Class
5. Click Save to collectively save all updates.

Adding Image to Lesson Plan Metadata

Step Screen
The image can be added in the Lesson Plan title.
1. Click the Edit Details link on the top left corner of the page.
2. Click the Image icon to add an image. The Select Image page appears
1.Click My images to view a list of all image files that you have previously, uploaded
2. Click All image to view a list of image files from the repository
Note: The repository contains all image files added and uploaded by any users. Choose an appropriate image file from any list.
3. You can add images by uploading an image from your local machine using Upload and use. A new Upload and use window appears.
To upload an image:
a. Select a file using Choose File and browse an image from your local drive or open the explorer in a separate window or drag and drop the required file on the screen
Note: Supported file formats are: .jpeg, .jpg, .png
b. Under Copyright and License, select if you want to:
# Make it available to everyone
# Not share with anyone
c. Click Upload and use images
4. Click Select to confirm the image
5. Click Cancel to discard the changes

Adding Content or Resources

Step Screen
To add content to the lesson plan
1. Click New Child to start adding a node to the newly created lesson
2. You can add another node at same level using New Sibling. Refer Adding Sibling Node to know more about adding New Sibling node.
3. You can add a node to the next level using New Child. Refer "Adding Child Node to know more about adding New Child node.
The LessonPlan Unit page appears
3. Click Add Resource from:
- The right bottom corner of the page
- Right click on the node name
- Use keyboard shortcut Ctrl+Alt+A
The Find & Select Activities page is displayed.
To Add Resource
1. Search the resource using search bar
2. Select the relevant resource cards from the available lists
3. Hover over the card to view the basic details about that resource
4. Click View Details to read illustrated information about the card in a new window.
a. Select Add to imbibe the resoure
b. Select Remove to remove the resource
5. Click Proceed to add resources to the section. The resource is added to the Resource section of the LessonPlan page.
6. Click Preview and Upload the Resources to know more.
7. Click Save to save the changes.
8. Click Send for Review to know more.

Preview and Upload the Resource

Step Screen
The Preview Resource page displays the Name, Description, Keywords related to the resource.
1. Click the play icon to load the added resource in the lesson plan
2. You can delete the resource using delete icon
3. Click the link in the breadcrumb to visit the corresponding lesson plan

Exploring What’s New

What’s New feature informs you about all the latest and significant changes build and made available for your better user experience.

Step Screen
1. Click on the What's New icon to view new functionalities and related content
Note: What's new icon appears in red color, when it is viewed for the first time. It turns to grey from next view, until next version of the feature is not available.
What's new page opens, with the highlights of the latest changes

Using Table of Contents Option

The Table of Contents tree seen in the left pane is used to navigate content compiled for a collection, textbook, course or lesson plan. You can manipulate the tree using either:

  • The burger menu associated with a tree node
  • Right-click options associated with a tree node, OR
  • Keyboard shortcuts

Adding Child Node

Step Screen
A child node is created as a node within the collection. Any number of child nodes can be created. To add a child node you can either:
1. Click New Child from the burger menu associated with the required node in the left navigation pane, OR
2. Right-click on the node name and select New Child, OR
3. Use the keyboard shortcut Ctrl+Alt+N, OR
4. Click New Child at the bottom of the navigation to add child node

Adding Sibling Node

Step Screen
A sibling node is created at the same hierarchy level as a child node. Any number of sibling nodes can be created. To add a sibling node you can either:
1. Click New Sibling from the burger menu associated with the required node in the left navigation pane, OR
2. Right-click on the node name and select New Sibling, OR
3. Use the keyboard shortcut Ctrl+Alt+Shift+N, OR
4. Click New Sibling at the bottom of the navigation to add sibling node

Node Metadata

Step Screen
On clicking New Child or New Sibling, the metadata is displayed on the right
1. Enter Unit Title
2. Enter Description
3. Add Notes to Teacher
4. Click Save to save the metadata
5. Click Add icon to add image to the metadata. For details on adding image refer, Adding Image to Lesson Plan Metadata
Note: The metadata for a child and sibling node is the same.

Deleting Content

Step Screen
1. Click Delete icon beside the burger menu
2. Click Yes to delete the added content
Note: You can alternatively delete content by clicking Delete in the burger menu drop-down

Reordering Content

Step Screen
1. Drag and drop the content to the desired position to rearrange the order of the content added to the lesson plan
1. Click the upward arrow to collapse the table of contents. Click the same arrow to expand collapsed table of contents.

Sending Lesson Plan for Review

Before the Lesson Plan is publihed, it must be reviewed. You can send it for reviewing and publishing.

Step Screen
1. Click Save on the header to save the lesson
2. Click Limited sharing to publish the lesson to the limited audiences
Note: Refer Limited Publishing and Sharing to view and share all the contents created and ready for sharing.
3. Click Send for review to send the created lesson for review
