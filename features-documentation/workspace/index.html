    My Workspace

    Sunbird provides you with a workspace that is only yours. Once you have your own unique profile, you can start using your workspace to create different types of content artifacts using the customizable content editor. You can save your drafts, send them for a review and rework on them as often as you like, until you publish it for use by the community.

    Accessing your Workspace

    1 Click Profile tab on the header,

    OR

    Click the Profile Icon and select Profile

    Image

    2 Click View your Workspace from the right side of the profile page

    OR

    Click Profile Icon on the header

    Select Workspace from the drop-down list

    Image

    Viewing your Workspace

    In your Workspace, you can:

    1. Create textbooks, courses, lessons and collections using the content editor tool

    2. View drafts of the content created

    3. Check the status of content that has been sent for review

    4. View published content

    5. View all uploaded content
