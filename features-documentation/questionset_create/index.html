Overview

To ensure that the content achieves the learning objective it is important to check and reaffirm the learner’s progress at different points in the study material. Adding questions and question sets is one method.

Step Screen
To add question set, click the Add Question Set icon on the toolbar. The question set browser page is displayed.
1. Click the Add Question Set icon on the toolbar

Selecting Questions - Using filter

Step Screen
1. Type keywords to search for resources.
2. Select Advanced Filters for better search result
 a) Language
 b) Difficulty
 c) Question Type
 d) Grade Level
 e) Concepts
3. Use My Questions slider to search for questions that you have uploaded. To search for questions available on the portal, disable this option.
4. The search result is displayed on the left pane
5. The selected question can be previewed on the right pane
6. Select questions from the results displayed by marking the checkbox
7. Click Create Question to create your own question
8. Click Next

Creating Questions

Step Screen
To create a question set
1. Click Create Question and select the type of question you want to create
2. You can create following types of question
 a) Creating Multiple Choice Question Set
 b) Creating Fill in the Blanks Question Set
 c) Creating Match the Following Question Set
3. Click Select
4.Click Cancel to cancel

Creating Multiple Choice Question Set

Step Screen
1. Enter the question. You can format the questions using rich text editor
2. Enter the answers
Note: Mark the correct answer amongst the multiple answers
3. Click Delete icon to remove an answer
4. Click Image icon to add an image
5. Click Audio icon to add an audio message
6. Click Refresh icon to preview your question
7. Select the layout from the drop-down menu
8. Use Shuffle Option to change the sequence of answers
9. Select Back to go back to the previous page
10. Click Next to enter the Question Details. The Create Question page appears

Creating Fill in the Blanks Question Set

Step Screen
1. Enter Question Set Title for the questionaire
2. The Max Score displays the maximum score for the question
3. Select the number of questions that should be displayed in each screen from the Display drop-down menu. The question amongst the selected questions is displayed in the right pane
4. Select Show Immediate Feedback to determine when the feedback for the answers should be displayed
5. Select Shuffle Question to enable shuffling of questions from the set
6. The list of selected questions appears in the left pane
7. Enter the score for each question using Configure Score
8. Click Add More Questions to increase the questions in the question set
9. Click Add to append the selected questions in Question Set. The Question Set is added on the slide.

Creating Match the Following Question Set

Step Screen
1. Enter the question
2. Enter the answers
3. Select the answer amongst the multiple answers
4. Click Delete icon to remove an answer
5. Click Image icon to add an image
6. Click Audio icon to add an audio message
7. Select Add answer to add another option in the Mark as Right multiple choice
8. Click Refresh icon to preview your question
9. Select the layout from the drop-down menu
10. Use Shuffle Option to change the sequence of answers
11. Select Back to go back to the previous page
12. Click Next to enter the Question Details.

Customizing Question Set

Step Screen
Enter Question Set Title
1. The Max Score box displays the maximum score allotted to the question
2. Select the number of questions that should be displayed in each screen from the Display drop-down menu
3.Use the Show Immediate Feedback slider to determine when the details for the answers should be displayed
4. Use the Shuffle Questions slider to enable shuffling of questions from the set
5. Click Add to add selected questions
6. Click Add More Questions to go back and select another question

Configuring Question Set

Step Screen
1. Click the trash icon to delete the added question set
2. Click the spanner icon to edit details of the question set. The settings options are displayed on the right
a) Click on the upward/downward arrows to determine the marks of the question set
3. Click the eye icon to view the preview of the question set
4. Click Done to save
