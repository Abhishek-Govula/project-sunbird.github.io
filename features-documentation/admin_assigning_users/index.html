One of the many responsibilities of the administrator on Sunbird is to assign roles and responsibilities to the various users within an organization. The administrator can assign more than one role to a user.

Prerequisites

Step Screen
1. You are logged in
2. You are currently on Profile page. You want to assign roles to different users in your organization

Selecting User

Step Screen
1. Select Users from the drop-down menu beside search bar
2. Enter name of the user
3. Press Enter or click the Search Icon.

User Search Results

Step Screen
1. The user details are listed on clicking search
2. For any registered user on Sunbird, you can:
 a) Assign Badges
 b) Edit User Role
 c) Delete User
3. Click on the user name to view the profile details of the user
4. Click Download to download details of all users uploaded. The details get downloaded in an excel file
5. Click Close icon to exit the search results display

User Profile

Step Screen
1. On clicking the user name, the following profile details can be viewed:
 a) Summary
 b)Address
 c) Experience
 d) Education
 e) Skill Tags
 f) Basic Profile
2. Click the close icon to go back to the list of users page
